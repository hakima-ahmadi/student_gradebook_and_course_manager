# Student Gradebook and Course Manager

## Full Name
Hakima Ahmadi

## Project Title
Student Gradebook and Course Manager

## Project Description
This project is a simple student gradebook system.
It can add students, add courses, enroll students,
record grades, calculate averages, and show student reports.

## How to Run
Run the main.py file.

## Classes
- Student
- Course
- Assessment
- Quiz
- Exam
- Project
- Gradebook

## OOP Features
- Encapsulation: Student and Gradebook classes use private attributes
  with getters and setters.
- Inheritance: Quiz, Exam, and Project inherit from Assessment.
- Method Overriding: Quiz, Exam, and Project
  override display_info() and grade_message().

## Custom Features
- Letter Grades (A, B, C, D, F)
- Teacher comment
- Dashboard (Shows total students, courses, and grade records)
